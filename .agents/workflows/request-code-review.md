# Requesting Code Review

## When to Use

Use after completing a major feature, before merging a branch, or when you want verification that work meets requirements.

## Process

1. **Read** `.agents/skills/request-code-review/SKILL.md` for the full pre-review checklist
2. Use the bundled agent prompt at `.agents/skills/request-code-review/agents/code-reviewer.md` for structured review
3. Complete the pre-review checklist before requesting review
4. Address all findings before claiming completion
